Pig Latin Translator


Pig Latin is a fun and simple word game or a secret code that many enjoy. No, it is neither Latin nor is it spoken by pigs. It simply involves taking English words and adding a little twist.

The exact origin of Pig Latin is unclear but it has been around for generations in different cultures, a beloved linguistic game for people of all ages.

I built this project using HTML, CSS, Boootstrap, and Javascript. You can see more in the Github repository. You can also check out my portfolio
